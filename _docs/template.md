## Listor

Numrerade listor

1. ett
2. två
3. tre

onumrerade listor. De kan skrivas på lite olika sätt, antingen med `-`, `*` eller `+`

- ett
- två
- tre

* ett
* två
* tre

+ ett
+ två
+ tre
